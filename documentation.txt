.gitignore 
    - скрывает файлы и папки от системы контроля версий Git

package.json 
    - хранит список пакетов, необходимых для проекта с нужными версиями, и на другой машине мы можем легко установить все пакеты, которые указаны там с помощью команды
    свойство private: true/false - приватный, открытый проект

    Скрипты:
        - /"scripts"/"start" - запускает/собирает приложение в режиме разработки
        - /"scripts"/"startWeb" - собирает неминифицированные файлы в папке dist
        - /"scripts"/"build" - запускает/собирает приложение в продакшн режиме
        - /"scripts"/"lint" - запускает eslint для проверки проекта на предмет стиль-гайд ошибок
        - "browserslist" - конфигурация для babel (для каких браузеров переводит)

webpack.config.js
    - хранит базовую конфигурацию webpack

.eslintrc
    - хранит базовую конфигурацию eslint

.eslintignore
    - скрывает файлы от eslint
